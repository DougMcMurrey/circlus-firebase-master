# Writing documentation

Using [Docsify](https://docsify.now.sh/).

You can add a caption to an image with the class `img-caption`.

You can float images or img-captions with classes:

- `right`
- `center`

```md
<div class="img-caption right">

![Associated Email](/_media/assoc_email_002.png)

Aute non voluptate veniam ad sint cillum. Ea dolore incididunt aliquip exercitation dolore pariatur non pariatur eiusmod amet. Ex magna elit occaecat mollit mollit et laboris ad labore. Aliquip minim fugiat elit consectetur ex Lorem ea. Sunt aliqua proident culpa aliqua.

</div>
```
